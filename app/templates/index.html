How it works

Three inputs. One weighted verdict.

01

Record the raw numbers

Teachers log attendance percentage, assignment scores, and exam scores for each student, every semester.

02

Set the weighting

Admins decide how much each factor should count — the weights always add up to 100%, validated automatically.

03

Get the prediction

The system calculates a weighted score and classifies each student — Top Performer, Average Performer, or At-Risk.
